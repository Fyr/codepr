Как работает плагин

1. Нужно подключить скрипты и стили:
			<link href="css/button-tabs.css" rel="stylesheet" type="text/css">
			<script type="text/javascript" src="js/jquery-1.10.2.min.js"></script>
			<script type="text/javascript" src="js/jquery.button-tabs.js"></script>
2. Сделать контейнер для вкладок с классом .button-tabs-links
			<div class="button-tabs-links">
				<a href="javascript: void(0)">Вкладка 1</a>
				<a href="javascript: void(0)">Вкладка 2</a>
				<a href="javascript: void(0)">Вкладка 3</a>
				...
			</div>
3. Рядом сделать несколько контейнеров для содержимого вкладок с классом .button-tabs-content.
Каждая вкладка соотв-ет своему контейнеру в порядке очередности. 
			<div class="button-tabs-content">Контент для вкладки 1...</div>
			<div class="button-tabs-content">Контент для вкладки 2...</div>
			<div class="button-tabs-content">Контент для вкладки 3...</div>
4. Поместить оба контейнера в общий элемент.
			<div class="button-tabs">
				...
				<div class="button-tabs-links">...</div>
				<div class="button-tabs-content">...</div>
				<div class="button-tabs-content">...</div>
				<div class="button-tabs-content">...</div>
			</div>
5. Применить для общего элемента плагин. Дополнительно можно задать опцию названия самой кнопки
			$(function(){
				$('.button-tabs').buttonTabs({
					title: 'Записаться на собеседование'
				});	
			});
6. Возможность менять цветовую схему реализована через поключение доп.стиля
			<link href="css/button-tabs-red.css" rel="stylesheet" type="text/css">
			или
			<link href="css/button-tabs-blue.css" rel="stylesheet" type="text/css">
